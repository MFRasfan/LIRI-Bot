# LIRI-Bot

This is a command line app that queries the 

Bands in Town API
spotify API
OMDB API

Returns to users information about their requested  Band/Songs/Movies

Commands

1 . concert-this

Run on Terminal 

node liri.js concert-this "BAND NAME"
